## The Elite CI/CD Suite (Core Features)
The primary value of this fork lies in its Super CI/CD Pipeline, integrating state-of-the-art DevOps practices and Hybrid AI analysis:

### Multi-AI Integration
*   Build Failure Analysis (Gemini): Real-time log diagnostics and remediation suggestions upon build failure.
*   Security Auditor (Groq Llama 3): High-speed vulnerability scanning of NuGet dependencies with risk summarization.
*   PR Intelligence (GitHub Models GPT-4o): Automated code review and architectural feedback for every Pull Request.

### Automated Quality Gates
*   Architecture Guard: Automated enforcement of MVP pattern constraints using NetArchTest.
*   Memory Leak Detection: Automated object retention testing via dotMemory Unit.
*   Mutation Testing: Code robustness verification using Stryker.NET.
*   Secret Protection: Automated credential scanning via Gitleaks.
*   Asset Logic: Automated resource size validation and image optimization scripts.

### GitOps Build Visibility
*   Persistent History: All AI analysis reports are permanently archived in a dedicated logs branch.
*   Historical Dashboard: Every commit is linked to a time-stamped report suite for full traceability.
*   Searchable Archives: Organized folder structure (reports/YYYY/MM/DD/...) for easy audit.

## Technical Stack
- Framework: .NET Framework 4.8 (WinForms)
- Database: SQLite
- CI/CD: GitHub Actions (Windows/Ubuntu hybrid)
- Versioning: Semantic auto-versioning via MinVer

## Setup
1. git clone https://github.com/AnhGam/Study-Document-Management.git
2. Open study-document-manager.sln in Visual Studio.
3. Required Repository Secrets: GEMINI_API_KEY, GROQ_API_KEY, DISCORD_WEBHOOK.
