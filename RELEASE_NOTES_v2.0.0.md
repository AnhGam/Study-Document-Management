## 🎉 Release v2.0.0 - Major UI/UX Overhaul & SQLite Migration

### ✨ Major Changes
- **SQLite Migration**: Chuyển từ SQL Server sang SQLite - ứng dụng hoạt động độc lập, không cần cài đặt database
- **Personal Mode**: Loại bỏ hệ thống đăng nhập, sử dụng đơn giản hơn
- **UI/UX Redesign**: Giao diện hoàn toàn mới với theme Teal/Emerald hiện đại
- **Toast Notification**: Hệ thống thông báo mới với DPI scaling support

### 🔧 Technical Improvements
- Responsive layout cho tất cả các form Management
- Dock-based layout thay thế TableLayoutPanel
- Consistent styling với AppTheme system
- Better DPI scaling support

### 📦 Downloads
- **StudyDocumentManager_v2.0.0_Setup.exe**: Bản cài đặt với wizard (khuyến nghị)
- **StudyDocumentManager_v2.0.0_Portable.zip**: Bản portable, giải nén và chạy trực tiếp

### 💻 System Requirements
- Windows 7 SP1 trở lên
- .NET Framework 4.8
